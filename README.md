# Sales Booster AI 🚀
*Developed for the Sinarmas Digital AI Day*

**Sales Booster AI** is an intelligent, agentic CRM built to empower insurance agents. By leveraging AI-driven lead scoring and contextual chat recommendations, it helps agents prioritize high-value prospects and close deals faster.

## ✨ Key Features
* **Intelligent Lead Scoring:** Automatically evaluates and ranks active leads (High, Med, Low) based on life events and demographic data.
* **AI Chat Assistant:** Context-aware chat interface powered by Claude to help agents draft personalized pitches and answer policy questions.
* **Agent Dashboard:** A dark-themed, intuitive UI tracking daily contacts, total coverage, and hot leads at a glance.

## 🛠️ Tech Stack
* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **AI/LLM:** Anthropic Claude API

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository
```bash
git clone [https://github.com/william-asj/sales-booster-ai.git](https://github.com/william-asj/sales-booster-ai.git)
cd sales-booster-ai
```

### 2. Install dependencies
```bash
npm install
```

### 3. Set up environment variables
Create a `.env.local` file in the root directory:
```bash
cp .env.example .env.local
```
Open `.env.local` and add your secure API keys:
```env
NEXT_PUBLIC_APP_URL=http://localhost:3000
CLAUDE_API_KEY=your_claude_api_key_here
```

### 4. Run the development server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the dashboard.
